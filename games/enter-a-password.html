Enter a Password




Tween

This game is rated tween because although, the game isn't supposed to be inappropriate, some of it's hidden websites may be seen as 'not appropriate for small children'.


Enter a password is just as it sounds, a funny little game where all you need to do is follow the instructions, (if it says "Enter a password" type in "a password" in the password field) solve a randomised puzzle and you'll be redirected to a built in website. Sounds boring I know but some of the websites you end up getting are a bit weird and funny (some of them might even contain an entire game within.)

Controls:

This game has no controls.
